## HTTP Requests CORS

## Purpose of Project

The purpose of this project is use different ways of sending different HTTP requests using Axios, SuperAgent, and fetch with CORS configuration enabled

## Quick Overview

This web application will utilize Axios, SuperAgent, and fetch to send HTTP requests to the server to validate user credentials.
